# Лабиринт сокровищ

Текстовая приключенческая игра, разработанная на Python.

## Описание

Игрок исследует древний лабиринт, собирает сокровища и разгадывает закадки.

## Установка 

make install

## Запуск

make project

## Проверка кода

make lint

## Сборка пакета

make build

## Разработка

Проект использует:

Python 3.12.10
poetry 2.2.1
ruff 0.14.6
Git 2.51.1
